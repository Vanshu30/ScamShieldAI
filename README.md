📌 Overview

ScamShieldAI is an AI-powered web application that detects phishing attempts, scam messages, fake banking alerts, and malicious URLs using Google's Gemini AI.

Users can paste suspicious messages or upload TXT files to receive an instant AI-powered security analysis.

⚙️ How It Works
User enters a suspicious message or uploads a TXT file.
Frontend sends the request to the FastAPI backend.
Backend analyzes the message using Google Gemini AI.
AI detects phishing indicators, suspicious URLs, and scam patterns.
Results are returned with:
Risk Score
Scam Type
Red Flags
Safety Recommendation
